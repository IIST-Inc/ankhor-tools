Command Reference

Shared command families for Ankhor / SASE CLI and API integrations.

Most user integrations should start from the scenario cookbook.

This page is for engineers who need command-level details.

secure

clear
help
quit / exit

Mode and Application Control

check-mode
change-mode <akm|pm|otp|byo>
is-activated
pin-status
set-pin <PIN>
change-pin <OLD> <NEW>
activate <PIN>
deactivate
reset-app

API Key Manager / AKM

akm list
akm add -t <Title> --key <API Key>
akm get -t <Title>
akm del -t <Title>

Password / Credential Manager / PM

pm list
pm add -t <Title> -n <Name> --password <Password>
pm get -t <Title> -n <Name>
pm del -t <Title> -n <Name>

OTP Manager

otp list
otp add --len <6..8> --ts <5..90> -t <Title> -n <Name> --secret "AA BB ..."
otp get -t <Title> -n <Name>
otp del -t <Title> -n <Name>

BYO Shared-Key Slots / Pairing

byo status
byo gen --idx <0..7> --pub <Hex64>
byo del --idx <0..7>
byo pair-init
byo pair --idx <0..7> --peer <Hex64>

BYO Shared-Slot Encryption

byo set-iv --iv <Hex16>
byo enc --idx <0..7> --pt <Hex32>
byo dec --idx <0..7> --ct <Hex32>
byo get-key --seed <ASCII>

BYO Edge-Key Encryption

byo edge-enc --pt <Hex32>
byo edge-dec --ct <Hex32>

BYO Local Secret Storage

byo secret-status
byo save --idx <0..3> --data <Hex>
byo read --idx <0..3>
byo del-secret --idx <0..3>

BYO Signing and Verification

byo sign-status
byo sign-gen --idx <0..7>
byo sign-del --idx <0..7>
byo sign --idx <0..7> --msg <Hex32>
byo verify --idx <0..7> --msg <Hex32> --sig <Hex64>
byo get-verify-key --idx <0..7>
